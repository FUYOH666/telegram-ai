## Описание

AI-GTD - это интеллектуальная система для автоматизации подготовки таможенных деклараций (ГТД) для таможенных брокеров в СНГ. Система автоматически извлекает данные из документов (инвойсы, упаковочные листы, контракты), определяет коды ТН ВЭД и генерирует черновик ГТД в формате XML.
